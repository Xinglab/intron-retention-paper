Please unzip Mus_musculus.GRCm38.91.chr.gtf.gz and mm10.fa.gz first in data folder.

The script provided is to annotate the effect (NMD) of intron for each transcript. 
We do not consider the last intron, because it would never cause NMD. 
For any details, please refer to our method section.

Run with: `python run_scripts.py`
